Implementation Details
======================

Module: backend/social-content/caption-validator.ts
- Exports validateCaption({ channel, text, hashtags? }) function
- Returns { ok: boolean, errors: string[] }

Platform Rules Implemented:
- Instagram (instagram_feed):
  * Max 2200 characters
  * Max 30 hashtags
  
- Facebook (facebook_feed):
  * Max 63206 characters
  * No hashtag limit

Error Codes:
- caption_empty: when text is empty
- caption_too_long: when text exceeds platform limit
- too_many_hashtags: when hashtag count exceeds platform limit (IG only)

Validation Approach:
- Simple, deterministic validation
- No external dependencies
- Supports multiple simultaneous violations
- Returns all errors in single call
